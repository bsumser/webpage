convolutional-neural-network

Convolutional Neural Networks are a class of neural network specializing in processing data with a grid topology, like images. Typically, they have 3 layers: convolution, pooling, and a fully connected layer.

Given an input size of W x W x D, with Dout number of kernels and spatial size of F, stride S and padding P, output size volume can be determined by this formula:

\[W_{out} = \frac{W - F + 2P}{S} + 1\].
